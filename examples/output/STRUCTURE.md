# Repository Structure

This repository is structured around an API service built using FastAPI, featuring agents that perform specific tasks related to code analysis and documentation generation. A testing suite is included, along with example usage scripts and user documentation.

## Technology Stack

FastAPI, Python, Docker

## Architecture

Microservices

## Components

- API: Handles network requests and routing with FastAPI framework
- Agents: Core functionalities for various agent operations
- Testing: Unit tests for the application to ensure functionality
- Examples: Example scripts showcasing the usage of the application
- Utilities: Helper functions and setups shared across the project
