# **Docpilot**
**AI-Powered Documentation Automation for GitHub**  

*Never let your docs lag behind code changes again. Auto-generate & sync docs on every push or PR.*

---

## **Why Docpilot?**  
- 🧠 **Context-Aware AI**: Analyzes code diffs, commit messages, and PR discussions using GPT-4/Claude-3.  
- ⚡ **Event-Driven**: Triggers doc updates on `push`/`merge` events.  
- 🔍 **Validation First**: AST parsing + unit test correlation ensure accuracy.

---

## **Features**  
| Feature | Description |  
|---------|-------------|  
| **Auto-API Docs** | Generates OpenAPI specs from REST endpoints |  
| **Changelog Drafts** | Converts PRs into structured release notes |  
| **Doc Health Monitor** | Flags outdated examples via GitHub Issues |  
| **Multi-Format Export** | Markdown, Swagger, Notion, Confluence |
